# Flexboard Backend

Backend service for Flexboard built with Bun.

## Prerequisites

- [Bun](https://bun.sh/) >= 1.0.0

## Getting Started

Install dependencies:

```bash
bun install
```

Run in development mode:

```bash
bun run dev
```

Run in production mode:

```bash
bun run start
```
